# JARVIS AI - Python Voice Assistant

A Python-based voice assistant inspired by Tony Stark's JARVIS.  
It automates tasks like WhatsApp messaging, web search, and system control.
It can listen to voice commands, speak responses, and automate tasks.

## Features

- Voice recognition
- Text-to-speech response
- Open applications
- WhatsApp automation
- Web search
- Modular command system

## Tech Stack

- Python
- SpeechRecognition
- Pyttsx3
- PyWhatKit
- Pygame

## Installation

1. Clone repository

git clone https://github.com/Clipperai/Mini-Jarvis-ai.git

2. Install dependencies

pip install -r requirements.txt

3. Run the assistant

python app.py

## Example Commands

- "Open YouTube"
- "Send WhatsApp message"
- "Search Python tutorials"
- "What time is it"

## Project Structure

app.py → Main controller  
listener.py → Voice input  
speak.py → Voice output  
commands/ → Task automation modules
